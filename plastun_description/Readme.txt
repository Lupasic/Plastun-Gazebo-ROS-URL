Название пакета: plastun_description


Задача пакета: Содержит конфиги для работы в rviz для:
1) Static (МРК в статике (можно повращать))
2) Gmapping (ручная постройка карты)
3) Move_base (Только автономная навигация)
4) Control_system (Конфиг для работы алгоритма работы)


Входные данные: для RVIZ Move_base /move_base_simple/goal 
Control_system /move_base_simple1/goal и /target_points


Выходные данные:


Составные части пакета:
1) launch - лаунч файлы
2) meshes - хранится модель МРК в формате NX, .stl
3) rviz - конфиги для rviz
4) urdf - описание модели на языке разметки urdf с использованием макро языка xacro


Параметры лаунч файлов:
У всех лаунч файлов одинаковый параметр - путь к rviz конфигу (основное различие)

Описание остальных файлов: Нет необходимости


Особенности: 
1) В meshes STL_min используется в модели, требуется модель уменьшать в 1к раз,
 относительно оригинала
