Название пакета: plastun_navigation


Задача пакета: Задачи автономной навигации МРК, а именно:
1) Gmapping (Построение карты)
2) Savemap (Сохранение, построенной вручную карту)
3) Navigation (Автономная навигация по построенной карте, сочетает amcl и move_base)


Входные данные: для Gmapping смотри документацию на пакет gmapping
для Savemap /map
для Navigation смотри пакеты доки amcl и move_base


Выходные данные: для Gmapping - /map
Savemap - .pgm и yaml 
для Navigation - cmd_vel


Составные части пакета:
1) launch - лаунч файлы
2) config - настройки для move_base, а так же amcl ( описаны в соответсвующих доках по ним)
3) maps - лежит описание и сама построенная карта(в эту директорию сохраняется карта,
используя Savemap)
