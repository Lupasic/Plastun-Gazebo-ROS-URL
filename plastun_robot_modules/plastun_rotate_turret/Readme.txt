Название пакета: plastun_rotate_turret


Задача пакета:  Реализована работа наклонно-поворотного устройства (НПУ) с помощью
а) Топика (задать название в лаунч файле)
б) action (для работы алгоритма)
Работа с НПУ происходит через UDP протокол.


Входные данные: углы поворота по вертикальной оси (yaw), и поперечной (pitch)


Выходные данные: команды приводам НПУ и action result


Составные части пакета:
1) action - хранит формат сообщения для action
2) launch - лаунч файл
3) src - код программы

Параметры лаунч файлов:
rotate_topic - топик для управления НПУ (без action)
ip - айпи адрес НПУ
port - порт
dest_port - порт назначения
Измерения поворота идет в тиках, поэтому нужно задать их
panMax - максимальный поворот по yaw
pan_zero - минимальный поворот по yaw
tiltMax - максимальный поворот по pitch
tilt_zero - минимальный поворот по pitch
pan_sign - знак поворота по yaw
tilt_sign - знак поворота по pitch
По углам
hor_max - макс поворот по yaw в радианах
hor_min - мин поворот по yaw в радианах
ver_max - макс поворот по pitch в радианах
ver_min - мин поворот по pitch в радианах
