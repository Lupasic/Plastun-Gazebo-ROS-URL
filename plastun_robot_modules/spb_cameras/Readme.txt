Название пакета: spb_cameras


Задача пакета: реализация работы ip камеры и получения ее настроек,
в "Особенности" ссылкак как настроить камеру


Входные данные: настройки сети для подключения к камере


Выходные данные: различные изображения (компрессионные или нет, цветные, черно-белые и другие)


Составные части пакета:
1) config - настройки камеры, полученные с помощью camera_calibration
2) launch - лаунч файлы
3) python - программный код на питоне


Параметры лаунч файлов: рассмотрим основной, cam
hostname - хостнейм
username - логин для айпи камеры
password - пароль для айпи камеры
prefix - префикс в названии
frame_id - для tf преобразований
camera_info_url - ссылка на файл с настройками, важно его задавать в таком формате 
package://[название пакета]/... или file://[полный путь]/...
image_transport - задаем какой вид изображений получаем (в компрессии или нет)
