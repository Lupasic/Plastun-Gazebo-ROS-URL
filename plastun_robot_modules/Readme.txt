Название пакета: plastun_robot_modules


Задача пакета:Содержит следующие пакеты управления отдельными частями МРК(есть запуск всего сразу)
1) bins - инерциальное измерительное устройство (imu)
2) common - различные драйвера для работы других устройств
3) lc_ros - отвечает за работу шасси
4) odo_tf_pub - публикация корректного улучшенного tf
5) plastun_activate_laser - активация лазерного целеуказателя, требуется запускать под sudo
6) plastun_modules_launches - настройки для активации модулей (в каждом пакете так же есть свои)
7) plastun_rotate_turret - работа с наклонно - поворотным устройством 
8) spb_cameras - активация работы камеры
