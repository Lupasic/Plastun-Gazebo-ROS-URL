Название пакета: plastun_activate_laser


Задача пакета: Реализована активация лазерного целеуказателя с помощью
а) Топика (задать название в лаунч файле)
б) action (для работы алгоритма)


Входные данные: либо action, передающий время активации лазера,
или activate_topic (параметр)


Выходные данные: командам для активации лазера и action result


Составные части пакета:
1) action - хранит формат сообщения для action
2) launch - лаунч файл
3) src - код программы


Параметры лаунч файлов:
1) activate_topic - название топика для активации целеуказателя
