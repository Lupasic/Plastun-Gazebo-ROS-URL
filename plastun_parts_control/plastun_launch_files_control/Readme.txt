Название пакета: plastun_launch_files_control


Задача пакета: содержит лаунч файлы, отвечающие за систему управления (global_control_system)

Входные данные: у каждого узла системы свои, подробнее в них


Выходные данные: у каждого узла системы свои, подробнее в них


Составные части пакета:
launch - директория с лаунч файлами


Параметры лаунч файлов: global_control_system
Для узла global_control_system: 
1) laser_duration - время работы лазерного целеуказателя
2) camera_info_topic - данные настроек камеры
3) target_points_topic - топик, передающий координаты цели
Для узла General_targeting:
1) lengh_to_cam_xy - смещение относительно центра вращения НПУ по xy
2) lengh_to_cam_yz - смещение относительно центра вращения НПУ по yz
3) target_height - высота цели
4) MRK_height - высота МРК до середины камеры
Для узла Image_detect:
1) cascad - путь к файлу с каскадом классификаторов
2) suscribe_camera_topic - топик, откуда считываются изображения
