Название пакета: plastun_global_control_system


Задача пакета: реализована логика работы алгоритма, то есть
помечена на карте цель, оператор ставит точку в RVIZ, откуда будет видна цель.
МРК добирается до цели, наводит первично целеуказатель, потом донаводится и активирует 
целеуказатель.


Входные данные: координаты цели (/target_points) и положение откуда будет видна цель 
(/move_base_simple1/goal). Так же необходимы настройки камеры (/wide_camera/camera_info)


Выходные данные: в итоге сообщение об успешности решения задачи
