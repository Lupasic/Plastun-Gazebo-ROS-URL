Название пакета: plastun_cascad_testing


Задача пакета: тестирование работоспособности обученного каскада.
в "Особенности" даны ссылки как обучать каскад классификаторов. 
Используется метод Виолы-Джонса, реализованный в OpenCV.


Входные данные: suscribe_camera_topic(параметр лаунч файла) - топик для приема видео


Выходные данные: /cascad_testing/output_video - вывод изображения с отмеченной целью


Составные части пакета:
1) cascads - обученные каскады классификаторов
2) launch - лаунч файл
3) src - программный код


Параметры лаунч файлов:
1) cascad" - путь к каскаду классификаторов
2) suscribe_camera_topic - топик для приема видео
3) image_transport - тип файла(сжатый или обычный)
